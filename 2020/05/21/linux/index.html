挂载光盘/DVD镜像文件

1.建立大文件
dd if=/dev/zero of=/srv/loopdev bs=1M count=512
2.大型文件的格式化.
mkfs.xfs -f /srv/loopdev
blkid /srv/loopdev
/srv/loopdev: UUID=”28bf9523-132b-4d65-a80b-98dfe331ab0c” TYPE=”xfs”
3.挂载.
mount -o loop UUID=”28bf9523-132b-4d65-a80b-98dfe331ab0c” /mnt
4.设置自动挂载.
vi /etc/fstab
/srv/loopdev /data/file xfs defaults,loop 0 0
5.mount -a

swap - 用磁盘来暂存放置内存中的信息,用到swap时候,主机键盘灯会闪个不停.

1.分区:使用gdisk在磁盘中分出一个分区槽给系统做swap.
2.格式化: mkswap [装置文件名] -> 将分区槽格式化为swap分区.
3.将swap装置启动. swapon [装置文件名].
4.通过swapon -s / free 观察内存的用量.

1.使用dd指令新建一个128MB的文件.
dd if=/dev/zero of=/tmp/swap bs=1M count=128
2.使用mkswap将/tmp/swap文件格式化为swap文件格式.
mkswap /tmp/swap
3.swapon 启动 /tmp/swap.
4.设定自动启用.
vim /etc/fstab
/tmp/swap swap swap defaults 0 0
ps:文件不要用UUID,系统根据UUID会查询块设备,不会查文件.
